# News Article Evaluation with NLP

this project is aimed to evaluating news articles using Natural Language Processing (NLP) techniques.
It allows users to enter a URL of a news article, which is then analyzed for sentiment polarity and subjectivity.

# How to Run the App 
1. Clone the repository to local machine.
2. Navigate to the project directory.
3. Install depandancies using `npm install`
4. Build the project using `npm run build`.
5. Start the development server using `npm run start`.
6. Open your web browser and go to `http://localhost:3000` to view the app.

## Dependencies
the project uses the following dependancies:
